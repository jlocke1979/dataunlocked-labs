

# MSDS 455 Assignment 1 – Life Flow Visualization

## How to Run

1. Navigate to the `app` folder:
   cd apps/life_flow/app

2. Start a local server:
   python3 -m http.server 8000

3. Open in browser:
   http://localhost:8000

## Description

This visualization explores organ transplant activity over time, moving from total transplants to major, minor, and rare categories. The narrative highlights how scale affects visibility and understanding of less common transplant types.
