# Documentation

## Data
The visualization uses cleaned OPTN / UNOS transplant and waitlist data. Dot counts are scaled approximations for visual comparison.

## Method
The chart uses D3.js to render organ categories as rows, with separate columns for waitlist volume, completed transplants, and approximate waitlist-to-transplant ratio.

## Design Notes
This revision is intentionally constrained to a one-page management-style visualization, following assignment feedback to prioritize clarity, citation, and concise interpretation.
