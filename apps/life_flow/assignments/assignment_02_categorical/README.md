# MSDS 455 Assignment 02 – Categorical Data Revision

## Overview
This submission presents a one-page categorical visualization comparing 2025 organ transplant waitlist demand with completed transplants by organ category.

## Main Finding
Organ demand exceeds completed transplants across categories, with kidney demand dominating the overall waitlist volume.

## How to View
From this folder, run:

python3 -m http.server 8000

Then open:

http://localhost:8000

## Files
- index.html: visualization entry point
- js/main_assignment_02.js: main D3 controller
- js/scenes_assignment2/: D3 visualization files
- data/: cleaned CSV data used by the visualization
- scripts/: Python data processing scripts

## Data Source
OPTN / UNOS national data reports, 2025.
