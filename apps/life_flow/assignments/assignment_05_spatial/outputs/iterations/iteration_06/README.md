# Iteration 06 — Single full-width volume maps (stacked)

Less busy alternative to Iteration 05: donor and transplant each get a full-width map, stacked vertically.

## Enable

In `app/main.js`:

```javascript
const LAYOUT_MODE = "dot_map_volume_singles";
```

Iteration 05 side-by-side layout:

```javascript
const LAYOUT_MODE = "dot_map_volume";
```

## Layout

1. **Donor Recovery Organizations** — conclusion subtitle, map, volume scale
2. **Transplant Centers** — conclusion subtitle, map, volume scale
3. Shared methodology note below both maps

Same data, palette, and sqrt bubble encoding as Iteration 05.
