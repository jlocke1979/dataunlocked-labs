# Venue Viability Lab

An experimental scenario model for testing whether a mid-sized event, performance, or civic venue can be financially sustainable.

This project is intentionally generic. It is not a valuation, recommendation, or investment memo.

## Core Question

What has to be true for a venue to work?

## Early Model Areas

- Ticketed events
- Artist/event costs
- Private rentals
- Nonprofit/community rentals
- Corporate rentals
- Sponsorships
- Grants/donations
- Fixed operating costs
- Break-even thresholds

## Status

Prototype / scenario sandbox.
