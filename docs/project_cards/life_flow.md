# Life Flow

A visual exploration of how organs move through the U.S. transplant system.

## Status

Graduate project for Northwestern MSDS 455. Submitted and currently frozen until grading is complete.

## Share link

Live project link:

`apps/life_flow/app/final_show.html`

## Short description

Life Flow uses interactive visualization to explore waitlists, donor supply, transplant centers, organ movement, and the human scale of transplantation.
