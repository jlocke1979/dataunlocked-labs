# DATAUnlocked Site Structure

## Current view

`dataunlocked-labs` is a lab/workshop repo, not the future polished website.

It contains experiments, coursework, apps, datasets, prototypes, and public demos.

## Future recommendation

Create a separate repo later:

`dataunlocked-site`

Purpose:
- polished public homepage
- curated project pages
- about page
- writing/blog links
- selected demos

## Proposed public site tree

- Home
- Projects
  - Life Flow
  - Golf Analytics
  - Uplands Garden Tracker
  - Other future projects
- Writing
  - Substack
- Ideas / Lab
  - Cognitive Observability
  - Medical Understanding
  - Decision Systems
- About
- Contact

## Relationship between repos

`dataunlocked-site`
- curated
- public-facing
- brand-first
- stable

`dataunlocked-labs`
- experimental
- code-heavy
- messy
- active workshop
- demos and prototypes

## Domain idea

`dataunlocked.com` should point to the polished site eventually.

Substack should become Writing / Blog, not the homepage.

GitHub Pages lab links can remain available as demos.
